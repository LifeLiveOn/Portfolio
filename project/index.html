Projects Showcase

RoofVision AI – Automated Roof Damage Detection

Developed an AI-powered computer vision system for roof damage detection using drone imagery, designed for State Farm agents. Integrated a customized RF-DETR (Region-Focused Detection Transformer) model to identify hail, wind, and surface defects, and automated report generation with analytical summaries and defect maps. (In production)

SmartPlay AI – LLM-Based EQ Training Platform

SmartPlay AI is an interactive LLM-powered platform that helps users improve emotional intelligence (EQ) through scenario-based Q&A challenges. Built with FastAPI, Cloudflare Tunnel, and LLM APIs, it delivers personalized feedback, contextual understanding, and real-time scoring across work, interview, and social themes.

VocalScribe – Voice-to-Text Converter

VocalScribe converts speech into text using Llama 3.1, Flask, and Jinja. It offers instant, accurate transcription for essays and presentations through a clean, user-friendly interface that enhances productivity and accessibility.

RTL Debugging with GNN + LLM – Research Project

Part of the UR2PhD research program, this project explores automating RTL code debugging using Graph Neural Networks (GNNs) and Large Language Models (LLMs). Contributed to model design, embedding fine-tuning, dataset preprocessing, and Retrieval-Augmented Generation (RAG) to enhance debugging precision. (In production)
